MK IT Task.

This is simple movie library web app.
The app uses http://www.omdbapi.com/ API for data , .net for backend part and reactjs for the frontend part.
The app uses JWT tokens for authentication.
Users can register/ add movies to favourites / add rating / add notes and also remove those.
